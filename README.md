# FinTrack — Controle Financeiro

Aplicativo Next.js para controle financeiro pessoal com autenticação Supabase, transações, categorias, metas e dashboard.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase

## Rodar localmente

```bash
npm install
npm run dev
```

Depois acesse:

```txt
http://localhost:3000
```
