✦ Segurança

Seus dados financeiros,
protegidos como num banco.

Levamos segurança a sério. Desde a infraestrutura até as práticas de privacidade — suas informações estão em boas mãos.

Criptografia AES-256
Todos os dados em repouso são criptografados com padrão bancário.
HTTPS + TLS 1.3
Tráfego protegido com o protocolo mais moderno disponível.
Backups diários
Seus dados são copiados automaticamente toda noite.
Zero venda de dados
Seus dados financeiros não são vendidos. Nunca. Para ninguém.
🔐
Supabase Infrastructure
Hospedado na infraestrutura Supabase com Row Level Security ativo em todas as tabelas.
💳
Pagamentos via Stripe
Processador de pagamentos de nível financeiro. Nunca tocamos em seus dados de cartão.
📋
Conformidade LGPD
Em conformidade com a Lei Geral de Proteção de Dados. Você tem controle total dos seus dados.
